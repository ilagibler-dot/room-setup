# Зал — планировка помещений

Отдельное мобильное приложение для расстановки залов: размеры в метрах, столы, стулья, рассадка гостей.

Не связано с Треком (event-crm). Данные хранятся локально в браузере (localStorage).

## Запуск локально

```bash
cd room-setup
npm install
npm run dev
```

Откройте [http://localhost:3001](http://localhost:3001)

## Деплой на отдельный сайт

### Vercel

1. Создайте новый проект в Vercel
2. Укажите **Root Directory**: `room-setup`
3. Framework: Next.js
4. Deploy

Можно привязать свой домен, например `zala.example.com`.

### Другой хостинг

```bash
cd room-setup
npm run build
npm run start
```

## Возможности

- Задание размеров помещения (м)
- Столы: круглый, квадратный, прямоугольный
- Элементы: бар, сцена, колонна
- Drag-and-drop / tap-to-place на мобильных
- Счётчик размера в метрах при расстановке
- Стулья вокруг столов
- Импорт списка гостей и назначение мест
- Экспорт рассадки в CSV
- PWA: можно добавить на домашний экран телефона

## Структура

```
room-setup/
  src/app/           — страницы
  src/components/    — редактор и UI
  src/lib/             — логика и localStorage
  public/              — PWA manifest и иконка
```
